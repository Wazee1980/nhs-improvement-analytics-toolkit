# SPC Guide for Healthcare Improvement

This guide provides a simple, practical introduction to using Statistical Process Control (SPC) charts in healthcare improvement work.

## Why SPC matters

SPC helps teams understand whether variation in a process is expected (common cause) or indicates a change (special cause).

This is critical when working on:
- Waiting list validation
- Length of stay reduction
- Clinic utilisation
- Discharge flow

## Common chart types

### Run Chart
- Plots data over time
- Useful for early-stage improvement work
- Helps identify shifts and trends

### XmR Chart
- More advanced than run charts
- Shows process stability using control limits
- Useful for operational performance monitoring

## Example measures

- Average length of stay (daily)
- Number of patients waiting over 52 weeks
- Clinic utilisation percentage
- Daily discharge volume

## Interpreting signals

Look for:
- Shifts (6 or more points above or below median)
- Trends (5 or more points increasing or decreasing)
- Points outside control limits (XmR)

These indicate a potential change in the system.

## Practical advice

- Use SPC alongside operational context, not in isolation
- Avoid reacting to single data points
- Combine with qualitative insight from teams
