# NHS Improvement Analytics Toolkit

A practical collection of templates, scripts, and frameworks to support healthcare improvement, operational performance, and system design.

This repository is designed for practitioners working on waiting list validation, flow improvement, clinic utilisation, and service transformation.

## What this includes

- SPC chart templates
- SQL and data extraction patterns for operational metrics
- Dashboard structures for elective care, urgent and emergency care, and mental health
- Process mapping templates
- Stakeholder mapping and system design tools
- Improvement frameworks used in delivery settings

## Why this exists

Across organisations, teams often rebuild the same tools from scratch. This repository aims to provide a shared starting point that can be adapted locally without duplicating effort.

The focus is on practical use rather than theory.

## Example use cases

- Validating elective waiting lists
- Monitoring clinic utilisation and productivity
- Reducing length of stay in inpatient settings
- Supporting discharge and flow improvement
- Designing learning and improvement networks
